# 快捷键
- 切换为`四视图` | Ctrl+Alt+q
- 复制并移动(进入`Grab模式`) ｜ Shift+d(+x|y|x 选择移动轴)
- 线/面挤出 ｜ e(+x|y|x 选择挤出方向轴)
- 上一步的操作应用到当前选择 ｜ Shift+r
- 打开模型右侧的小工具栏(位置插件功能等调整) ｜ n
- 填充(两点连成线,线圈填成面) ｜ n
